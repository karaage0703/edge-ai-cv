# セットアップ

AI環境セットアップ。

## ファイル内容

| File  | Description |
|:-|:-|
| install_opencv.sh  | OpenCVのセットアップ |
| install_pytorch.sh  | PyTorchのセットアップ |
| install_tensorflow.sh  | TensorFlowのセットアップ |

## 使い方

### OpenCV セットアップ

```sh
$ cd ~/edge-ai-cv/scripts
$ ./install_opencv.sh
```

### ONNX Runtime セットアップ

```sh
$ sudo pip3 install onnxruntime
```
